This repository contain a python analysis on the King County Housing market.

The file Business-Pres contains a presentation on the results with some notes.
In the folder picture, several pictures that are used in the presentation are stored.

The file Immersive_Data_ScienceImmersive_Data_Science_Project1_EDA.ipynb contains a Jupyter notebook with the King County Housing market analysis

The King_County_House_prices_dataset.csv contains the to analyze dataset.
Column_list.md contains information about the dataset.

Instructions.ipynb contain instructions about the project.
